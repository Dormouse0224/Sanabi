컴포넌트 관련

오브젝트 관련
 - 오브젝트를 레벨에 추가하는 기능
 - 오브젝트와 프리펩 간 전환


1. Mesh
 - 이미지에서 드래그로 프리뷰 카메라 조작하는 기능
 - 프리뷰 토폴로지를 변경시키는 기능


2. Material


3. Texture


4. Sound
 - 소리를 미리 들어보는 기능


5. Graphic Shader
 - 셰이더의 각종 옵션(토폴로지, 렌더 도메인, 각 스테이트)을 변경하는 기능
 - 그래픽 셰이더는 각 셰이더 파일 내부에 어떤 파이프라인 단계가 어떤 함수 이름을 가지는지 컨테이너에 저장하고,
 에셋에서 셰이더를 컴파일 할 때 컨테이너에 저장된 정보를 이용해 컴파일하도록 한다.


6. Compute Shader
 - 컴퓨트 셰이더의 각종 수치 조정 기능
 - 컴퓨트 셰이더는 구조체를 통해 입력받을 텍스쳐 또는 상수 데이터를 UI로 전달한다.
 - 컴퓨트 셰이더는 각 셰이더에서 컴파일 해야 할 함수 이름을 컨테이너에 저장하여 에셋 관리자가 컴파일할 때 컨테이너의 
 데이터를 통해 컴파일 할 수 있도록 한다.


7. Prefab
 - 기존 오브젝트를 프리펩으로 만드는 기능
 - 프리펩을 레벨에 오브젝트로 배치하는 기능


8. Sprite
 - 새로운 스프라이트를 생성하거나 기존 스프라이트를 편집하는 기능


9. Flipbook
 - 플립북에 씬을 추가하거나 제거하는 기능
 - 플립북의 각 씬 속성을 편집하는 기능


10. Transform comp


11. Camera comp


12. Collider2D comp
- PhysX 와 역할이 중복되므로 삭제


13. PhysX comp
 - 액터의 각 물리적 속성을 편집하는 기능
 - 충돌 이벤트 설정 기능


14. FSM comp
 - 각 상태와 상태 전환 조건을 편집하는 기능
 - State 스크립트를 FSM 에 등록하는 기능


15. Level
 - 레벨을 새로 생성하고 오브젝트를 추가하는 기능


16. Button comp
 - 버튼에 연결할 함수를 설정하는 기능


17. State
 - FSM 컴포넌트와 연계
